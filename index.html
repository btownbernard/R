Benchmark test

We saw before that over two-thirds of stops were of black drivers. The by-race stop counts are only meaningful, though, when compared to some baseline. If the Philadelphia population was about two-thirds black, then two-thirds of stops being of black drivers wouldnt be at all surprising.

population_2016 %>% 
  mutate(prop = num_people / sum(num_people))
## # A tibble: 5 x 3

Data Visualization

###Bar Chart

#data prep

barprep <- stops %>% 
  group_by(subject_race) %>% 
  summarize(
    n = n(),
    prop = n / nrow(.)
  )
#make the plot

bar <- ggplot(barprep,
       aes(x=reorder(subject_race,n), y=n))+
  geom_bar(stat="identity", 
           position="identity", 
           fill="blue")+
  geom_hline(yintercept = 0) +
  labs(title="Stopped Drivers by Race",
       subtitle = "WhiteAmerican drivers got stopped the most in the city of Wichita, Kansas")+
  coord_flip()

options(scipen=10000)
